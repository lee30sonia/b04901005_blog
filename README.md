## HW5: blog
### usage:
$cd blog/ <br>
$npm install <br>
#please have mongodb installed (db version v3.4.7) <br>
$mongod --dbpath ./db <br><br>

#open another terminal <br>
$cd blog/ <br>
$node setup.js <br><br>
$node server.js <br><br>

#open still another terminal <br>
$cd blog/ <br>
